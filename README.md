## 📄 简介

为`LiteLScript`开发者准备的`自动补全`和`代码提示`的`TypeScript.d.ts`文件

## 🔧 使用

### 搭配 VSCode 插件 `LLScriptHelper` 使用

将本存储库的`src`文件夹内的所有文件&文件夹放到 LLScriptHelper 插件设置里的补全库固定存放地址的 `JS` 文件夹内即可

### 单独使用

请在你编写的插件文件最顶上添加一行注释

```js
/// <reference path="[本仓库根目录路径]/src/index.d.ts" />
```
